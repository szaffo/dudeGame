## Directory structure guide

```
A következő struktúrájú mappaszerkezetet
kell létrehozni a szerveren:

src
   model
      *.java
   view
      *.java
   controller
      *java
   images
      *.gif
      *.jpg
   sounds
      *.wav
doc
   SoftwareModel.eapx (Enterprise Architect modell)

Csak ilyen fájlokat szabad feltölteni:
.java, .gif, .jpg, .wav, .eapx
```
